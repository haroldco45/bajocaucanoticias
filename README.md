# 🗞️ Bajo Cauca Noticias
**La voz del Nordeste Antioqueño**

Revista digital y telediario con IA activa para la subregión Bajo Cauca de Antioquia, Colombia.

## ✨ Funcionalidades

- 📰 **Revista editorial** con diseño telediario oscuro y ticker de noticias en tiempo real
- 🤖 **IA activa (Claude Sonnet)** — genera noticias verosímiles de la región por categoría
- 🎙️ **Chat de voz** — reconocimiento de voz en español colombiano + síntesis de voz
- ⏰ **Reloj Colombia** en tiempo real (UTC-5)
- 🗂️ **Categorías**: Todo, Seguridad, Economía, Comunidad, Cultura, Deportes, Minería
- 📲 **PWA** — instalable en Android/iOS, funciona offline
- 📣 **Open Graph** — vista previa en WhatsApp y redes sociales

## 🚀 Despliegue en GitHub Pages

1. Crear repositorio: `bajocauca-noticias` en github.com/haroldco45
2. Subir `index.html` y `README.md`
3. Ir a **Settings → Pages → Branch: main → /root**
4. URL activa: `https://haroldco45.github.io/bajocauca-noticias/`

## 🏠 Despliegue en Hostinger

1. Entrar al **hPanel** de Hostinger
2. Ir a **Administrador de archivos → public_html**
3. Crear carpeta `bajocauca`
4. Subir `index.html`
5. URL: `https://vibraspositivashm.com/bajocauca/`

## 🛠️ Stack técnico

- HTML5 / CSS3 / JavaScript Vanilla
- Claude API (claude-sonnet-4-20250514)
- Web Speech API (reconocimiento + síntesis de voz)
- PWA: Service Worker + Web App Manifest
- Single-file — sin dependencias externas
